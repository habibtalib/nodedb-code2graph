- Matrix's 37-language list (`rust` through `haskell`, all 22 pre-existing + all 15 candidates that landed in 01-01/01-02) is generated by an `awk`/`grep` command against `Cargo.toml`'s `[features]` block, not hand-typed — verified byte-identical to the derived list via a diff check

- Confirmed `lint`, `test`, and `bindings` jobs are byte-for-byte unchanged (`git diff --stat` shows only additions)
- Fixed a real pre-existing bug in `src/resolve/symbol_table.rs`: 4 test functions unconditionally imported `SqlExtractor`/`HclExtractor`, which fails to compile under `cargo check --tests`/`cargo test --no-default-features --features <lang>` for any non-sql/non-hcl language — the same isolation-bug shape as the `luau` bug 01-01 fixed.
